Deep-Learning Based Chest X-Ray interpretation with LLM-Driven Radiology Reports

Summary: An ML based software that interprets an X-Ray image, checks for 3 diseases, namely, Lung Carcinoma, Pneumonia and Emphysema, and creates a medical report detailing the rationale behind the findings.

Objectives:

  • Train a CNN model that can detect lung cancer, pneumonia and emphysema in an X-Ray image and distinguish between an affected and clean chest X-ray.
  • Reduce processing time to improve detection time and RTAT (Report Turnaround Time) by using an LLM to generate a descriptive report
  • Improve performance of the CNN to minimize false negatives

Software Used:

    Dataset: NIH Chest X-ray14 CNN Framework: DenseNet201 LLM: LlaVA Deployment: Streamlit
